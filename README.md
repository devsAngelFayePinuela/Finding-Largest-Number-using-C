# Finding Largest Number in C

A simple C program that accepts numbers from the user and determines the largest value using conditional statements.

## Features

- Accepts user input
- Compares numbers
- Displays the largest number
- Beginner-friendly implementation

## Technologies Used

- C Programming Language
- GCC / MinGW / Code::Blocks / Dev-C++

## How to Run

1. Clone this repository.
2. Open the project in your preferred C IDE or compiler.
3. Compile the source file.
4. Run the program.

## Sample Output

Input:
```
Enter first number: 15
Enter second number: 27
Enter third number: 12
```

Output:
```
The largest number is: 27
```

## Learning Objectives

This project helps practice:
- Variables
- User input (`scanf`)
- Conditional statements (`if`, `else if`, `else`)
- Comparison operators
